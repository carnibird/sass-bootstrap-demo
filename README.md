# sass-bootstrap-demo

## Etapes pour reproduire le repository

### Installer node

https://nodejs.org/en/download/

### Créer le répertoire du projet

sass-bootstrap-demo
aller dans le répertoire

### Initialiser npm

En ligne de commande:

```bash
npm init
```

### Install Sass via npm

En ligne de commande:

```bash
npm install --save-dev sass
```
